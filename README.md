# Tennis Scorer

A personal Android app for keeping score of a tennis match from the notification
shade. Score with one tap per point (`+ player`, `undo`) without leaving whatever
you're doing on your phone. Match setup lives in the app; live scoring lives in a
persistent, interactive notification.

Single match at a time, no accounts, no cloud, live scoring only.

Default format: no-ad pro set (first to 8 games, 7-point tiebreak at 7–7).
Other formats are selectable per match.
